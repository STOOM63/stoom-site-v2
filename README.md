# STOOM Site V4 Client

Site multi-pages prêt à publier :
- Accueil épuré
- Nouveautés
- Économies tabac
- Calculateur DIY
- Concours
- Débuter
- Contact

Aucune interface admin n'est incluse dans cette version. Les contenus se modifient dans :
- `data/nouveautes.json`
- `data/concours.json`

Pour publier :
1. Décompresser le ZIP
2. Glisser tout le contenu dans GitHub
3. Activer GitHub Pages sur `main` et `/root`
